# RN-multi-select

A React Native multiple select example using typescript

## Deployment

To deploy this project first run metro server

```bash
  npm start
```
To run on Android

```bash
  react-native run-android
```
To run on IOS

```bash
  react-native run-ios
```

## Used in the project

A cool library to implement icon usages.
